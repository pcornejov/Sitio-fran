# Sitio web — Francisca Arriagada

Sitio de servicios de contabilidad, tributación y administración de personal en San Carlos, Región de Ñuble, Chile.

## Stack

HTML/CSS/JS estático, sin build step. Desplegado en GitHub Pages vía GitHub Actions (`.github/workflows/deploy.yml`) en cada push a `main`.

## Estructura

- `index.html`, `servicios.html`, `sobre-mi.html`, `contacto.html`
- `assets/css/styles.css`, `assets/js/main.js`, `assets/img/`

## Desarrollo

Este sitio no requiere build: para previsualizar localmente basta con abrir los archivos HTML en un navegador o servir la carpeta con cualquier servidor estático (ej. `python3 -m http.server`).
